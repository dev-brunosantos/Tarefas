# APP TAREFAS

## CRIAÇÃO
O App foi desenvolvido com React Native usando o EXPO.

## OBJETIVO

<div>
  O Objetivo do projeto é facilitar o controle de tarefas diárias. </br>
  Com o APP TAREFAS o usuário poderá administrar as taferas que devem ser feitas diáriamente.
</div>

## O QUE O APP FAZ ?

* Tela de LOGIN / Tela de CADASTRO </br>
  - Todo usuário poderá criar seu prórpio login e ter acesso apenas as suas tarefas
* Tela HOME </br>
  - Ver tarefas
  - Criar tarefas
  - Editar tarefas
  - Excluir tarefas
